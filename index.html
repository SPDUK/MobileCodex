What the skill actually adds

It does not replace Codex. It changes Codex's defaults for away-from-laptop engineering.

Run safe commands

Codex should run safe local commands itself instead of asking you to run them from a laptop.

Create safe previews

When web output exists, Codex should verify locally, start ngrok, extract the real URL, and report the tunnel status.

Capture browser proof

Visual work should include mobile screenshots and browser evidence, not a vague "looks good."

Return compact handoffs

Final replies should include result, preview or blocker, proof, and the next useful move.
